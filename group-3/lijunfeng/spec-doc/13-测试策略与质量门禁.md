## 3. 质量门禁

### 3.1 准入条件

| 检查项 | 标准 |
|--------|------|
| 代码编译 | 无编译错误 |
| API 文档 | Swagger UI 可正常访问 |
| 基础功能 | 页面可正常加载 |

### 3.2 准出条件

| 检查项 | 标准 |
|--------|------|
| 功能完成度 | P0 功能 100% 完成 |
| 测试通过率 | 所有测试用例通过 |
| 性能达标 | 首屏加载 ≤ 3 秒 |
| 数据完整性 | 3 月初至今数据完整 |

← 03 §4 成功标准

## 4. Bug 分级

| 级别 | 定义 | 处理时限 |
|------|------|----------|
| P0-致命 | 系统崩溃，核心功能不可用 | 立即修复 |
| P1-严重 | 功能异常，影响使用 | 24 小时内 |
| P2-一般 | 小问题，不影响主流程 | 下次迭代 |
| P3-建议 | 优化建议 | 评估后决定 |
