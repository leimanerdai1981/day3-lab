## 2. 组件规格

### 2.1 Timeline（时间轴）

| 属性 | 说明 |
|------|------|
| 展示形式 | 水平时间轴，每周为一个节点 |
| 节点内容 | 周编号 + 局势分值 |
| 分值颜色 | 0-30: 绿色(低风险), 31-70: 黄色(中风险), 71-100: 红色(高风险) |
| 交互 | 点击节点切换当前展示周 |
| 选中状态 | 当前选中节点高亮显示（边框加粗） |

← 04 §4 R-01, R-02, R-06

### 2.2 SituationCard（局势卡片）

| 属性 | 说明 |
|------|------|
| 展示内容 | 当前选中周的局势分值 + 局势动态摘要 |
| 分值展示 | 大字体数字，带颜色背景 |
| 摘要展示 | 文本区域，最多显示 3 行 |
| 日期范围 | 显示当前周的开始和结束日期 |

← 04 §4 R-02

### 2.3 ThematicFundCard（主题基金卡片）

| 属性 | 说明 |
|------|------|
| 展示内容 | 主题名称 + 基金名称 + 涨跌幅 |
| 布局 | 横向排列，每行最多 5 个卡片 |
| 涨跌幅颜色 | 正值绿色，负值红色 |
| 主题列表 | 原油、黄金、煤炭、电力、天然气 |

← 04 §4 R-03

### 2.4 SmallScaleFundCard（小规模基金卡片）

| 属性 | 说明 |
|------|------|
| 展示内容 | 基金名称 + 规模 + 涨跌幅 |
| 布局 | 横向排列，每行最多 5 个卡片 |
| 规模展示 | 显示 "规模: X.X亿" |
| 涨跌幅颜色 | 正值绿色，负值红色 |
| 交互 | 点击卡片弹出净值趋势图 |

← 04 §4 R-04, R-09

### 2.5 FundNavTrendModal（基金净值趋势弹窗）

| 属性 | 说明 |
|------|------|
| 触发方式 | 点击主题基金或小规模基金卡片 |
| 展示内容 | 基金名称 + 累计净值趋势图 |
| 时间范围切换 | 支持 1个月/3个月/6个月/1年 切换 |
| 图表类型 | 折线图，展示 accumulated_nav 走势 |
| 关闭方式 | 点击遮罩层或关闭按钮 |

← 04 §4 R-09, R-10

## 3. 交互行为

### 3.1 页面加载

1. 加载时间轴，默认选中最新一周
2. 加载当前周的局势分值和摘要
3. 加载当前周的主题基金数据
4. 加载当前周的精选小规模基金数据

### 3.2 切换周节点

1. 用户点击时间轴上的某周节点
2. 该节点高亮显示
3. 局势卡片、主题基金、小规模基金数据同步更新
4. 页面无刷新，数据平滑过渡

← 05 §2 US-004 验收标准

### 3.3 数据加载状态

| 状态 | 展示 |
|------|------|
| 加载中 | 显示骨架屏或 Loading 动画 |
| 加载成功 | 正常展示数据 |
| 加载失败 | 显示错误提示，提供重试按钮 |

## 4. 响应式适配

| 屏幕宽度 | 布局调整 |
|----------|----------|
| ≥1200px | 默认布局，时间轴横向展开 |
| 768px-1199px | 基金卡片每行 3 个 |
| <768px | 基金卡片每行 2 个，时间轴可横向滚动 |

← 03 §3 范围边界：仅 Web 端，但做基本适配

## 5. 错误处理

| 错误场景 | 处理方式 |
|----------|----------|
| API 请求失败 | 显示错误提示，提供重试按钮 |
| 数据为空 | 显示 "暂无数据" 占位图 |
| 网络断开 | 显示网络错误提示 |
| 基金无净值数据 | 显示 "暂无净值数据" 提示 |

## 6. 交互流程

### 6.1 查看基金净值趋势

1. 用户在主题基金或小规模基金区域点击任意基金卡片
2. 弹出净值趋势图弹窗，默认展示近 3 个月数据
3. 用户可切换时间范围（1m/3m/6m/1y）
4. 趋势图展示累计净值走势
5. 用户点击关闭按钮或遮罩层关闭弹窗

← 05 §2 US-005 验收标准
